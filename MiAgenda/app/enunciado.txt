Enunciado del proyecto:

Nombre de la app: MiAgendaApp

Descripción general:
La aplicación MiAgendaApp permitirá al usuario organizar sus tareas diarias de manera eficiente, con opciones para seleccionar la hora de una actividad mediante un TimePickerDialog, elegir sus preferencias con Checkbox y RadioButton, y gestionar varios elementos de la interfaz, como botones e imágenes opcionales para mejorar la experiencia.

Funcionalidades principales:

Pantalla de inicio con botón de "Agregar tarea":
La pantalla principal tendrá un botón que al presionarlo llevará a una nueva actividad donde el usuario podrá agregar una nueva tarea.

Formulario para agregar tarea:

Un campo de texto para ingresar el nombre de la tarea.

Un campo TimePickerDialog para seleccionar la hora de la tarea.

Dos botones de tipo RadioButton que permitirán al usuario elegir la prioridad de la tarea (por ejemplo, "Alta", "Media", "Baja").

Varios CheckBox que permitirán seleccionar opciones adicionales (como "Tarea repetitiva", "Requiere notificación", etc.).

Un Button para guardar la tarea.

Opcionalmente, la app permitirá agregar una imagen a la tarea, donde se puede elegir una imagen desde la galería o desde un recurso en la app.

Lista de tareas:
La pantalla principal mostrará una lista de las tareas agregadas, con un RecyclerView. Cada ítem en la lista tendrá:

El nombre de la tarea.

La hora de la tarea.

La prioridad seleccionada (a través de los RadioButton).

Un icono o imagen (si el usuario agregó una).

Un botón para editar o eliminar la tarea.

Requerimientos técnicos:

Java como lenguaje de programación.

Android Studio como entorno de desarrollo.

Uso de TimePickerDialog para seleccionar la hora de cada tarea.

Uso de RadioButtons para seleccionar la prioridad de la tarea.

Uso de Checkboxes para opciones adicionales de la tarea.

Uso de RecyclerView para listar las tareas agregadas.

Opción opcional para agregar imágenes a las tareas (puede ser a través de un ImageView y selección de imágenes desde la galería).
